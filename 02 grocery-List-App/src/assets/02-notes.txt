Notes from my buddy GPT

  Think of it this way:

SetGroceryItems([
  ...groceryItems,
  { name: inputValue, quantity: 1, completedStatus: false },
]);
groceryItems is an array, for example:
[
  { name: "Milk" },
  { name: "Eggs" }
]
...groceryItems spreads (expands) the array into its individual elements:
{ name: "Milk" },
{ name: "Eggs" }
So JavaScript sees this:
[
  { name: "Milk" },
  { name: "Eggs" },
  { name: inputValue, quantity: 1, completedStatus: false }
]
Finally, SetGroceryItems() replaces the old state with this new array.
Why is this spread?

Because you're expanding one array into its elements.

A quick rule to remember:

Spread (...) = Expand one array/object into its contents.
Rest (...) = Collect multiple values into one variable.
